# Bank Customer Churn Analysis | Power BI Dashboard

## 📊 Dashboard Preview
![Bank Churn Dashboard](./dashboard.png)

## 📌 Executive Summary
[cite_start]This project features a comprehensive **Churn Analysis Dashboard** built to help a banking institution identify why customers are leaving[cite: 2, 3]. By analyzing a robust dataset of **10,000 records**, this report visualizes the relationship between customer demographics, financial standing, and retention rates to drive data-backed business decisions.

## 📈 Key Performance Indicators (KPIs)
* **Total Records Analyzed**: 10,000
* [cite_start]**Target Churn Rate**: 15.00% (Benchmarked for performance tracking) [cite: 5]
* [cite_start]**Customer Retention**: Detailed analysis of active vs. inactive membership status [cite: 12]

## 🔍 Deep-Dive Insights
* [cite_start]**Demographics**: The customer base is **55.92% Male** and **44.08% Female**[cite: 10, 11, 28].
* [cite_start]**Geography**: Data is strategically split across **France (39.96%)**, **Germany (39.76%)**, and **Spain (20.27%)**[cite: 17, 18, 19, 21, 63].
* [cite_start]**Activity Status**: Only **36.08%** of the customer base is currently **Active**, while **63.92%** are **Inactive**[cite: 12, 13, 34].
* [cite_start]**Age Groups**: The highest concentration of customers is in the **41-50** age bracket[cite: 46, 47].
* [cite_start]**Credit Scores**: A majority of customers fall within the **601-700** credit score range[cite: 61, 62].
* [cite_start]**Account Balances**: Customers with balances between **100k - 200k** represent a critical segment for high-value retention efforts[cite: 81, 82].

## 🛠️ Technical Stack & Skills
* **Tool**: Power BI Desktop
* **Data Modeling**: Implementation of a Star Schema for efficient reporting and filtering
* [cite_start]**DAX (Data Analysis Expressions)**: Created custom measures for **Churn Rate**, **Total Customers**, and **Target Benchmarks**[cite: 38, 54, 67].
* [cite_start]**Data Transformation**: Used **Power Query** to group raw data into "Age Groups" and "Credit Score" tiers for better trend visualization[cite: 29, 36, 47].
